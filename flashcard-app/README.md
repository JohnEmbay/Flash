# Micro-Learning Flashcards

## Problem Statement
Users struggle to retain newly learned vocabulary due to lack of quick, repeatable study sessions. This app solves the "Micro-Learning Retention Gap" by providing a lightweight flashcard tool for 5-minute learning sessions.

## Solution Overview
A responsive web app built with HTML, CSS, and vanilla JavaScript that enables users to:
- Add vocabulary flashcards
- Study in randomized order with flip animations
- Track mastery progress
- Persist data with localStorage

## Features
✅ **Core Features**
- Add flashcards (term + definition)
- Study mode: flip cards, next, mark learned
- Shuffle/randomize cards
- Progress tracking (% mastered + visual bar)
- Responsive design (mobile-first)

✅ **Enhancements**
- localStorage persistence
- Smooth flip animations (CSS 3D transforms)
- Reset progress button
- Clean, modern UI with gradients

## Technologies Used
- HTML5 Semantic Elements
- CSS3 (Flexbox, Grid, Animations, Gradients)
- Vanilla JavaScript (ES6+ Classes, DOM, localStorage)
- No frameworks or external dependencies

## How to Run
1. Open `/workspaces/Flash/flashcard-app/index.html` in your browser
2. Or use Live Server extension in VS Code
3. Add flashcards and start studying!

## Usage
1. **Add Cards**: Enter term/definition in the form
2. **Study**: Auto-switches to study mode
3. **Controls**: Flip, Next, Mark Learned, Shuffle, Reset
4. **Progress**: Updates automatically (persists in browser)

## CRUD Operations Implemented
- **Create**: Add new flashcards
- **Read**: Display current card (term/definition)
- **Update**: Mark as learned
- **Delete**: Optional (not implemented to encourage review)
